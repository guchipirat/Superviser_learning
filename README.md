# Описание проекта
Интернет-магазин «В один клик» столкнулся со снижением активности покупателей. Привлечение новых клиентов перестало быть эффективным, так как большая часть целевой аудитории уже знает о магазине. Для решения проблемы было предложено разработать решение, которое позволит персонализировать предложения для постоянных клиентов с целью увеличения их покупательской активности.
Задача: построить модель машинного обучения, прогнозирующую снижение покупательской активности клиентов на основе их поведения и характеристик.


# Используемые модели и методы
В ходе исследования были обучены и протестированы следующие модели классификации:
- KNeighborsClassifier (K-ближайших соседей)
- DecisionTreeClassifier (Дерево решений)
- LogisticRegression (Логистическая регрессия)
- SVC (Метод опорных векторов)

Для поиска оптимальной модели был создан пайплайн, включающий:

- Различные методы масштабирования данных;
- Перебор моделей;
- Подбор гиперпараметров с помощью кросс-валидации.

# Результаты
Лучшая модель
По итогам экспериментов наилучший результат показала модель KNeighborsClassifier с гиперпараметром n_neighbors = 17.

# Бизнес-выводы и рекомендации
В ходе анализа данных и моделирования были получены важные инсайты для бизнеса:

- Акции и активность: чем выше доля товаров, купленных во время акций, тем выше вероятность снижения активности. Это может указывать на недостаточное количество проводимых акций. Рекомендуется увеличить частоту акционных предложений.
- Поведение на сайте: пользователи, просматривающие больше категорий товаров за визит, демонстрируют более высокую активность. Это подтверждает необходимость разработки системы рекомендаций для увеличения вовлеченности.
- Премиум-сегмент : наблюдается отток клиентов с премиум-подпиской. Они редко участвуют в стандартных акциях. Рекомендуется разработать специальные предложения и дополнительные функции именно для премиум-клиентов.
- Популярные категории: наибольший спрос наблюдается в категориях «Товары для детей» и «Косметика и аксессуары». Эти категории следует активнее использовать в персонализированных рекомендациях.
- Группа риска: клиенты, просматривающие мало страниц за визит, имеют повышенную вероятность снижения активности. Для них необходима система рекомендаций, стимулирующая более глубокое изучение каталога.
